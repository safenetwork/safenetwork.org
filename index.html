The SAFE Network is a secure data storage and communication platform.

It's an open source project and it was started by a Scottish company called MaidSafe.

Core libraries

Previously, the SAFE Network was being developed in C++.

Now, all the core libraries are written exclusively in Rust, a systems programming language that runs blazingly fast, prevents nearly all segfaults, and guarantees thread safety.

SAFE Network apps

The SAFE Network enables you to build scalable, decentralized applications.

To access the SAFE Network, your apps can communicate with SAFE Launcher via a JSON API.

SAFE Pods

MaidSafe is committed to help catalyze collaboration and cooperation by setting up developer pods in various places around the world.

These “SAFE Pods” will focus on distributing core development and bring together creators of applications which will run on top of the SAFE Network.
