SAFE Browser

Download SAFE Browser

Browse public content and web applications on the SAFE Network (Local or Shared Section) using the SAFE Browser. Further information on connecting to the Shared Section, including a step-by-step guide can be found here.

Accounts can be created using the SAFE CLI and the Authenticator daemon.

Please be aware that MaidSafe might need to restart the Shared Section, wiping all data that is stored on it. They will announce this in a dev update (when possible) prior to taking it down.
